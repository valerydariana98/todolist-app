# TodoList App

Aplicación Fullstack moderna para gestión de tareas y listas personalizadas.

Cada usuario podrá autenticarse con Google, crear múltiples listas de tareas y administrar tareas de manera organizada. Además, el proyecto integrará subida y descarga de archivos utilizando Google Drive API.

---

# Objetivo del Proyecto

Desarrollar una aplicación Fullstack aplicando conceptos reales de desarrollo Backend y Frontend:

- Arquitectura MVC
- APIs REST
- Autenticación JWT
- Google OAuth
- MongoDB
- Integración con APIs externas
- Deploy Fullstack

El proyecto busca simular una aplicación real de productividad similar a herramientas como Google Tasks, Trello o Notion en una versión simplificada.
